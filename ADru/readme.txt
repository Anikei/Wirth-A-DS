Чтобы играть с примерами, нужно запустить Блэкбокс.
При первом запуске Блэкбокса откроется документ ADru/Справка.odc, в котором есть гиперссылки для всех примеров, а также дополнительные объяснения.

NB Если выходить из Блэкбокса по команде меню File, Exit, то ББ будет запоминать конфигурацию открытых окон и восстанавливать ее при следующем запуске. При выходе по Alt-F4 такого запоминания не происходит.
